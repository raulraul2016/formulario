crearemos un Formulario de Registro en un sitio de Artistas. El formulario debe cumplir con lo siguiente:

Poseer un título: Registro.
Tener un campo Nombre de tipo texto.
Tener un campo Tipo que permita seleccionar entre tres opciones de una lista: “Músico”, “Pintor” o “Actor”.
Tener un campo Comentario de tipo texto 
Tener un botón de Continuar.

En esta oportunidad nos centraremos en  las validaciones de los campos de un formulario.

Seguiremos trabajando con el Formulario de Registro del sitio de Artistas para aplicar validaciones a los campos de dicho formulario. Para ello, coloca los siguientes campos obligatorios:

Nombre
Tipo
Comentario
Acepta Términos y Condiciones
Recuerda colocar un asterisco  *  para indicar visualmente al usuario que el campo es obligatorio
